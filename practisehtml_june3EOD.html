Summary of table

  1. table tag for creating the table
  2. tr for table row, th for table heading which get displayed as column name, td for table data
  3. border attribute--- for table th and td. it gives border to the table and the table cells. even if we specify as solid that it should display as single line between cells it will be displayed as double lines between cells
  4. border-collapse -- so when it comes as double lines btwn cells, to make it display as solid.the borders will combine into one.so should be used when table,th,td --> border and border-collapse
  5. cell padding -- gives space a gap btwn cell data and its border.so it should be given to th and td
  6. text-align --- gives alignment to the text in left,right and center. give it for th,td.
  7. th will be by default bold and centered
  8. border-spacing --gives spaces btwen cells in a table and shld not be used when border-collapse attri is used. so mention it for table
  9. colspan -- to combine 2 or more columns. rowspan -- to combine 2 or more rows
  10. caption tag -- to add caption to a table
  11. to give spl styles to a table give id to the table
  12. we can specify more styles using css. suppose if alternate rows have to be displayed in white and gray bgcolor then use table#t01 tr:nth-child(even)
  13. colgroup tag and inside there are col tags..so these tags are used to specify a particular style to specific columns. use colspan for col tags, if colspan=2 then first 2 column will have that style
  14. there are other tags for table as well like thead,tfoot,tbody
  15. thead for headings of the column names of the table.for th,tr inside thead.
  16. tfoot will be displayed in the last row of the table
