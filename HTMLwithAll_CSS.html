Block level elements

block level elements start on a new line. any other ele after them also starts on a new line

  • p
  • h1 to h6
  • ol, ul
  • pre
  • address
  • blockquote
  • dl
  • div
  • fieldset
  • form
  • hr
  • noscript
  • table

Inline level elements

inline level elements can appear within any sentences and do not begin on new line. inline level elements form short structures than block level elts

  • b,big,i,small
  • abbr,cite,code,dfn,em,kbd,strong,samp,var
  • a, bdo, br, img, map, object, q, script, span, sub, sup
  • button, input, label, select, textarea
